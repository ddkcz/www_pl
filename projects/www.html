PRJ-011 CADsmart 2026

Własna strona internetowa

Rola: Deweloper

Projekt i wykonanie wizytówki w sieci — od zera do działającego portfolio. Dzięki intensywnemu wykorzystaniu AI, cały serwis powstał z rąk inżyniera mechanika bez wcześniejszego doświadczenia we front-endzie.

Stack i narzędzia

  • Vanilla HTML / CSS / JavaScript — bez frameworków.
  • Hosting na Cloudflare Workers z automatycznym deploymentem przez GitHub Actions.
  • Dynamiczne ładowanie layoutu (header i footer) ze wspólnego pliku.
  • System tagów i filtrów projektów — single source of truth w JS.
  • Obsługa trybu ciemnego i jasnego z persystencją w localStorage.
